ESPOSTA

ESPOSTA is a software product intended to serve as a tool to combat the spread of fakenews. It is capable of calculating and presenting quantitatively the reliability of a news that is shared with the application. In addition, it allows generating an image with the result of the verdict, allowing it to be shared on the networks where the original information circulates.

Built with: Flutter, .NET Core 3.1, C#
